# Cyole's Dotfiles

Personal dotfiles managed with [dotter](https://github.com/SuperCuber/dotter).

## Included Configurations

- **Shell**: zsh, fish, starship
- **Terminals**: kitty, ghostty, foot
- **Editor**: neovim
- **Window Manager**: niri
- **System Tools**: btop, fastfetch
- **Others**: fcitx5, fontconfig, solaar, DankMaterialShell

## Usage

```bash
# Install dotter
paru -S dotter

# Deploy configurations
dotter deploy
```
